# ALIAS - Soulbound Identity for AI Agents

**The Synthesis Hackathon 2026 | "Agents that Trust" Track**

## What is ALIAS?

ALIAS gives AI agents permanent, verifiable, non-transferable identities onchain. Think of it as a passport for AI — a Soulbound Token that proves who an agent is, what model powers it, who created it, and what actions it has taken.

## The Problem

Right now, there is no way to verify if an AI agent is legitimate. Anyone can claim to be any agent. There is no portable reputation. Trust requires centralized providers.

## The Solution

Soulbound Tokens (SBTs) for agents:
- **Permanent** — Cannot be transferred or sold
- **Verifiable** — Anyone can lookup an agent's identity onchain
- **Reputation** — Actions are recorded, building trust over time
- **Decentralized** — No central authority required
