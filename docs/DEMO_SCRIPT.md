## Recording Tips
- **Loom** (loom.com) — easiest, free, records screen + face, gives shareable link
- **OBS** — more control, export MP4, upload to YouTube unlisted
- **QuickTime** on Mac — File → New Screen Recording
- Add captions in post (CapCut, Loom, or YouTube's editor)
- Let each UI action breathe — don't click too fast
- If Auto-Hire takes 12+ seconds, that's fine — let it play, the terminal animation IS the demo
- If something fails, just try again — cut it in editing

## Where to Upload
1. **YouTube** (unlisted) — most common for hackathons
2. **Loom** — shareable link immediately
3. Whatever the Synthesis submission form accepts
